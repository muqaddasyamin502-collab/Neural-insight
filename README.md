# 🧠 Neural Insight — AI Portfolio

A single-page, dark-themed portfolio website built to showcase my journey and projects as an Artificial Intelligence student. Designed with a futuristic, AI-inspired aesthetic and a fully interactive **AI Decision Simulator** — no external API required.

🔗 **Live Demo:** [muqaddasyamin502-collab.github.io/Neural-insight](https://muqaddasyamin502-collab.github.io/Neural-insight/)

---

## ✨ Features

- 🎨 Dark, futuristic UI with an AI/tech-inspired visual theme
- 🤖 Interactive **AI Decision Simulator** — simulates AI-style decision-making directly in the browser, with no backend or API calls
- 📱 Fully responsive layout (mobile + desktop friendly)
- 🧩 Sections for About, Skills, Projects, and Contact
- ⚡ Lightweight — single HTML file, no build steps or dependencies

```
Neural-insight/

```

## 🚀 Running Locally

No installation or build steps needed.

```bash
git clone https://github.com/muqaddasyamin502-collab/Neural-insight.git
cd Neural-insight
```

Then simply open `index.html` in your browser — or use a local dev server:

```bash
# Option 1: just double-click index.html
# Option 2: use VS Code's Live Server extension
# Option 3: python -m http.server
```
